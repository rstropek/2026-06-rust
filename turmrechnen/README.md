# "Turmrechnen"

Einen "Zapfen" rechnen.

* Startwert 3
* Höhe 5

```txt
  3 * 2 = 6
  6 * 3 = 18
 18 * 4 = 72
 72 * 5 = 360
360 / 2 = 180
180 / 3 = 60
 60 / 4 = 15
 15 / 5 = 3
```

Level 1: Businesslogik, Ausgabe, sonst nichts
Level 2: Trennung Logik und Ausgabe
Level 3: Unit Tests hinzufügen
Level 4: Formatieren
